# Psychometric estimation of LLM output

This directory holds psychometric measurement of LLM behavioral output —
inference scripts' raw outputs, scoring/IRT analysis, and knitted reports.
Complements the internal-representation track (RepE / probing / steering)
worked on in `rgb_reports/` and `scripts/`.

## Layout

- **`gfc_tirt/`** — Graded forced-choice (Okada GFC-30) + Thurstonian IRT.
  Current direction (since the 2026-04-11 pivot away from Likert). Includes
  Stan models (`tirt_okada*.stan`), per-model and pooled fits, neutral
  baselines, fake-good condition, and the Haiku 4.5 + 4 RGB-model lineup.

- **`soft_evidence/`** — Pre-pivot Likert work. BFI / IPIP-NEO collected as
  5-point Likert with full per-option logprobs; soft-evidence GRM / NRM IRT
  analyses; Llama-2 vs Llama-3.2 comparison. Methodological signature is
  IRT on logprob distributions (the soft-evidence likelihood) rather than
  one-hot top-1 responses.

## Running

R analyses are written as R Markdown (`.Rmd`) — knit with `rmarkdown::render()`
or in RStudio. Stan models are fit by their corresponding `fit_tirt_*.R`
drivers and produce `.rds` files (archived separately, see RAW_ARTIFACTS.md).

GFC inference is generated by `scripts/run_gfc_anthropic.py` (Anthropic) or
`scripts/run_gfc_ollama.py` (Orin / local Ollama). The per-model-per-condition
JSONs in `gfc_tirt/` are the raw outputs of those scripts.
