# Raw artifacts archive

Large fitted objects, raw collected pickles, and full-size CSVs are not
committed to this repo. They live in the **`big5_results`** archive repo
(private, no public mirror):

    git@github.com:statisfactions/big5_results.git    (branch: main)

This `psychometrics/` tree was imported from `big5_results` via
`git subtree add --squash` from the `slim` branch (which excludes large
binaries). The `main` branch of `big5_results` retains the full history
including all artifacts listed below â€” clone it directly when you need a
specific fit.

## Regen recipe

Every fit listed above can be regenerated by running its driver script in
this tree. Drivers consume the JSON inference outputs that ARE committed
here, so no external data is needed beyond the archive repo for checkpoint
inspection.
